# Simulations Numériques - Stage Astraea Marine

Ce dépôt contient les codes que j'ai réalisé à l'aide d'une IA durant mon stage chez ASTRAEA MARINE.

Ces interfaces ont été conçues pour appuyer la conception et le dimensionnement d'un prototype d'USV (Unmanned Surface Vessel).

## 📂 Contenu du dépôt

Le dépôt est composé de trois outils principaux :

1. **simulation de flexion (poutre).html** : Interface de calcul de RDM (Résistance Des Matériaux) pour le dimensionnement de la poutre centrale en carbone. Elle permet d'évaluer la flexion et les contraintes maximales.
2. **Simulation de levage (pignong).html** : Modélisation cinématique et calcul des efforts pour le système de déploiement par pignon-crémaillère.
3. **Simulation de stabilité (coques).html** : Outil d'analyse de la stabilité de l'USV et calcul des limites de flambement des cloisons en fonction des charges appliquées.

## 🚀 Utilisation

Ces outils sont développés en HTML et ne nécessitent aucune installation particulière. 

Pour les utiliser :
1. Téléchargez le fichier souhaité (ou clonez le dépôt).
2. Ouvrez le fichier `.html` directement dans n'importe quel navigateur web moderne (Chrome, Firefox, Edge, etc.).
